异能者(ERAZER)无线机械键盘GK501

发表于 2025-05-29 分类于 daily Waline：

连接模式设置

1. 模式切换

  • 系统切换: 拨动侧键选择 Mac/Win 系统（Win系统下 Win键与 Alt键位置互换）。
  • 连接模式切换: 拨动侧键选择 USB有线/OFF/无线模式。

2. 蓝牙连接（支持3台设备）

  1. 切换至对应系统（Mac/Win）。
  2. 拨动开关至无线模式。
  3. 长按 FN+Q/W/E 5秒进入配对（对应蓝牙1/2/3），按键闪烁。
  4. 设备搜索并连接 “ERAZER GK 501”，成功后停止闪烁。
  • 切换设备: 按 FN+Q/W/E 切换已配对的设备。

3. 2.4G连接

  1. 切换至对应系统（Mac/Win）。
  2. 拨动开关至无线模式。
  3. 长按 FN+R 5秒进入配对，R键闪烁。
  4. 插入接收器，连接成功后停止闪烁。

4. USB有线连接

  1. 切换至对应系统（Mac/Win）。
  2. 拨动开关至有线模式。
  3. 插入数据线，T键闪烁3次后连接成功。

功能操作

1. 电量管理

  • 查询电量: 无线模式下按 FN+B，数字区亮红灯显示电量（如“1、2、3”亮表示30%）。
  • 低电量提示: ESC键闪烁，电压低于3.1V自动关机。

2. 背光控制

  • 亮度调节: FN+↑/↓（共5档）。
  • 速度调节: FN+←/→（共5档）。
  • 模式切换: FN+HOME 循环切换（如波浪、呼吸、烟花等22种模式）。
  • 开关背光: FN+Backspace。

3. 快捷键功能

Windows系统：

组合键功能组合键功能
FN+F1打开“我的电脑”FN+F7播放/暂停
FN+F2搜索FN+F9静音
FN+F3计算器FN+F10音量-
FN+F5上一曲FN+F11音量+

Mac系统：

组合键功能组合键功能
FN+F1屏幕亮度-FN+F8播放/暂停
FN+F2屏幕亮度+FN+F12音量+

4. 其他功能

  • 恢复出厂设置: 长按 FN+ESC 3秒。
  • 睡眠模式: 无操作2分钟进入浅睡眠，30分钟后深度睡眠（按任意键唤醒）。

指示灯说明

  • 状态灯: CAPS/WIN/NUM 无功能时亮，有功能时灭。
  • 连接模式灯: 底部显示蓝牙1/2/3、2.4G、有线状态。
  • 充电提示: 充电时指示灯灭，充满恢复灯效。

注意事项

  1. 蓝牙/2.4G配对失败后，键盘保留最后一次成功配置。
  2. 接收器需插入电脑USB口使用，配对时确保距离小于10米。
  3. 双系统模式下，仅背光和按键功能恢复默认，系统设置不变。
